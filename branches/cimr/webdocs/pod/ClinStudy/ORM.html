NAME

ClinStudy::ORM - An object-relational mapping layer used by the ClinStudy::Web application to interact with its database.

SYNOPSIS

    use ClinStudy::ORM;
    my $schema = ClinStudy::ORM->connect( $dsn, $user, $pass, $args );
    my @patients = $schema->resultset('Patient');

DESCRIPTION

The ClinStudy::ORM ORM is generated automatically from the SQL schema used to create the underlying ClinWeb database. See the DBIx::Class::Schema::Loader manpage module documentation for details.

SEE ALSO

the ClinStudy::Web manpage, the DBIx::Class manpage
