NAME

ClinStudy::Web::TestCalc - Aggregate test calculator base class

SYNOPSIS

 use Moose;
 BEGIN { extends 'ClinStudy::Web::TestCalc' }

DESCRIPTION

This module forms a base class for all the custom aggregate test calculator modules (e.g. BILAG, BVAS etc.).

METHODS

check_result_existence

A convenience method which checks for the existence of a given test result attached to a given Visit. Arguments are the ClinStudy::ORM object, the Visit object and the test name. Return value is the number of specified test results attached to the container object.
