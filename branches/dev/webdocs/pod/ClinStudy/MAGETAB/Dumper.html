NAME

ClinStudy::MAGETAB::Dumper - MAGE-TAB export from the ClinStudy database.

SYNOPSIS

 use ClinStudy::MAGETAB::Dumper;
 my $dumper = ClinStudy::MAGETAB::Dumper->new(
     uri => $database_home_uri,
 );

 # Add annotation for each of the files being included.
 foreach my $file ( @filenames ) {
    $dumper->add_file( $file );
 }

 # Write out the MAGE-TAB files.
 $dumper->dump();

DESCRIPTION

Module created to facilitate the export of MAGE-TAB documents from the ClinStudy database, primarily using the web-based JSON API.

ATTRIBUTES

uri

The main URI of the ClinStudy web application (for example, when running under the Catalyst test server this would be http://localhost:3000 by default).

username

The username to use in connecting to the database.

password

The password to use in connecting to the database.

builder

A Bio::MAGETAB::Util::Builder object. This will be created automatically by default; this option should not typically be used unless you know what you're doing.

METHODS

add_file

Add a new filename to the nascent Bio::MAGETAB model.

dump

Dump out the constructed MAGE-TAB documents using the Bio::MAGETAB::Util::Writer modules.

SEE ALSO

the Bio::MAGETAB manpage, the ClinStudy::XML::Dumper manpage
