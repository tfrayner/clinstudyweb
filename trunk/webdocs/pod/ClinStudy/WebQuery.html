NAME

ClinStudy::WebQuery - CIMR::QueryObj class accessing ClinStudyWeb via REST API (DEPRECATED).

SYNOPSIS

 use ClinStudy::WebQuery;
 my $qobj = ClinStudy::WebQuery->new(
     uri        => 'http://localhost:3000',
     username   => 'my_username',
     password   => 'my_password',
     id_field   => 'assay_barcode',
 );
 my @values = $qobj->query( $id, @queryterms );

DESCRIPTION

This is a simple module which has largely been superceded by the ClinStudyWeb R package. It is a subclass of CIMR::QueryObj and is intended to support the same usage. See the CIMR::QueryObj manpage for more information on the API. Note that the queryterms attribute is set for you by this subclass.

ATTRIBUTES

uri

The main ClinStudyWeb uri to use for REST access.

username

Your ClinStudyWeb username.

password

Your ClinStudyWeb password.

METHODS

query

See the CIMR::QueryObj manpage for a description of the query API.

SEE ALSO

the CIMR::QueryObj manpage
