NAME

ClinStudy::XML::TabReannotator - Reannotation of tab-delimited files.

SYNOPSIS

 use ClinStudy::XML::TabReannotator;
 my $builder = ClinStudy::XML::TabReannotator->new({
     tabfile => 'filename.txt',
 });
 my $root    = $builder->root();

 # Writes to STDOUT.
 $builder->read();

DESCRIPTION

A module used to connect to a ClinStudy database and pull out missing attributes to fill in a partially completed tab-delimited data file.

ATTRIBUTES

Note that XML TabReader attributes are handled by the ClinStudy::XML::TabReader superclass.

database

The ClinStudy::ORM object to use for database queries.

METHODS

read

Quick wrapper method which just reads in the header row and spits it out in the appropriate column order (i.e. alphabetically sorted). The method then hands off control to the read method in the superclass (see the ClinStudy::XML::TabReader manpage).

SEE ALSO

the ClinStudy::XML::Builder manpage, the ClinStudy::XML::TabReader manpage
