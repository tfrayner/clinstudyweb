NAME

ClinStudy::Web::TestCalc::BVAS - Calculation of vasculitis BVAS scores

SYNOPSIS

 use ClinStudy::Web::TestCalc::BVAS;

DESCRIPTION

This TestCalc module is designed to convert the per-system scores defined by the original BVAS scale (Luqmani R.A. (1994) Birmingham Vasculitis Activity Score (BVAS) in systemic necrotizing vasculitis. QJM. 87(11):671-8. PubMed ID: 7820541) into two numerical indices ("BVAS1" and "BVAS2" in the database). The BVAS1 index relates to new or worse illness, while BVAS2 reflects persistent symptoms. A final overall score is easily calculated by adding these two indices together; at present this calculator does not do this.

METHODS

calculate

The method used by the ClinStudy::Web::TestCalc superclass to create a new aggregate test result in the database, or update the old result. See the superclass documentation for more details.
