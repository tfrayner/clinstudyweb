NAME

ClinStudy::Web::Controller::User - Catalyst Controller

DESCRIPTION

Catalyst Controller.

METHODS

index

Dummy action; currently unused.

list

Generate a listing of local database users.

add

Add a new user account to the database; this is the public action used by new users to set up their initial account prior to being assigned a user group. This method does not grant any intrinsic access privileges.

edit

Edit a user in the database. This is an admin-level operation.

modify

Edit one's own user profile in the database. This action is open to any database user.

reset

Reset one's own password in the database. Alternatively, admin-level users may reset the passwords for any user.

delete

Remove a user from the database. Note that the audit history tables link to the user tables, but there are no database constraints preventing user removal.

create_password

Creates a random alphanumeric password of the specified length (default is eight characters).

reset_user_password

Sets the password of the specified user to the given string. If no string is supplied, create_password is used to generate one.

load_form

Load the HTML::FormFu form config. We use this in preference to the FormConfig attributes because it allows us to automatically insert our DBIC object onto each form's stash.
