NAME

ClinStudy::Web::Controller::Rest - Catalyst Controller for the REST API

DESCRIPTION

Catalyst Controller.

METHODS

begin

Initial method called during a REST query. This method handles the authentication of the user.

sample

A wrapper action for queries on Samples which redirects to the appropriate GET or POST method.

sample_GET

Query Samples by the supplied name attribute and return the appropriate serialized hashref to the user.

assay_file

A wrapper action for filename-based queries on Assays which redirects to the appropriate GET or POST method.

assay_file_GET

Query Assays by the supplied filename attribute and return the appropriate serialized hashref to the user.

assay_barcode

A wrapper action for identifier-based queries on Assays which redirects to the appropriate GET or POST method.

assay_barcode_GET

Query Assays by the supplied identifier (barcode) attribute and return the appropriate serialized hashref to the user.

dump_sample_entity

Given a Sample object from the database, construct a hashref suitable for passing back to the user.

dump_assay_entity

Given an Assay object from the database, construct a hashref suitable for passing back to the user.
