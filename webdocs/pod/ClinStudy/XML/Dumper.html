NAME

ClinStudy::XML::Dumper - XML export from ClinStudyDB databases.

SYNOPSIS

 use ClinStudy::XML::Dumper;
 my $dump = ClinStudy::XML::Dumper->new(
     database => $db_schema,
 );

 # Export a list of DBIx::Class::Row objects.
 my $doc  = $dump->xml(\@patients);
 
 # Export all data.
 my $full = $dump->xml_all();

DESCRIPTION

Module handling the generation of XML from a ClinStudyDB database.

ATTRIBUTES

See the ClinStudy::XML::Export manpage for attributes defined in this superclass.

METHODS

xml

Method which, when passed a list of either Patient or AssayBatch database objects, will generate the appropriate in-memory XML::LibXML tree structure appropriate for output with toString().

row_to_element

Special-cased method which handles TestResult, VisitEmergentGroup and PatientPriorGroup appropriately. See the ClinStudy::XML::Export manpage for documentation on the superclass method.

rel_to_attr
