NAME

ClinStudy::XML::Schema - Class for validating XML against a schema

SYNOPSIS

 use ClinStudy::XML::Schema;
 my $loader = ClinStudy::XML::Schema->new(
     schema_file => 'my_schema.xsd',
 );
 $loader->validate($doc) or die("Incorrect XML");

DESCRIPTION

This class is a generic validator module for XML which checks that it conforms to a given schema.

ATTRIBUTES

schema_file

The XML Schema file against which to check the XML instance document. Either this, or the schema attribute is required.

schema

An instance of the XML::LibXML::Schema class, pointing to the XML Schema to use for validation. Either this, or the schema_file attribute is required.

METHODS

validate

Validate the supplied XML::LibXML::Document object against the stored schema. Returns true on success, false otherwise.

EXPORT

None by default.
