NAME

ClinStudy::XML::AdminLoader - Import user metadata into a ClinStudy database.

SYNOPSIS

 use ClinStudy::XML::AdminLoader;
 my $loader = ClinStudy::XML::AdminLoader->new(
     schema_file => 'my_schema.xsd',
     database    => $db_schema,
 );
 $loader->validate($doc) or die("Incorrect XML");
 $loader->load($doc);

DESCRIPTION

A module designed to handle validation of XML according to the ClinStudyAdmin schema, and loading of such XML into a database. This is a database-specific subclass of the ClinStudy::XML::Import class.

METHODS

load_element

Special-cased method handling TestPossibleValue and RelatedVocab elements appropriately. See the ClinStudy::XML::Import manpage for documentation on the generic load_element method.

load

Special-cased method handling RelatedVocab elements appropriately. See the ClinStudy::XML::Import manpage for documentation on the generic load method.

the ClinStudy::XML::Import manpage, the ClinStudy::XML::Schema manpage
