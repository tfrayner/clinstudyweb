NAME

ClinStudy::XML::TabReader - Utility module for creating ClinStudyML documents.

SYNOPSIS

 use ClinStudy::XML::TabReader;
 my $builder = ClinStudy::XML::TabReader->new({
     tabfile => 'filename.txt',
 });
 my $root    = $builder->root();

 $builder->read();

 # Output the finished document.
 $builder->dump( \*STDOUT );

DESCRIPTION

A module initially intended to facilitate conversion of tab-delimited data to ClinStudyML. See the ClinStudy::XML::Builder superclass for more information.

ATTRIBUTES

Note that XML Builder attributes are handled by the ClinStudy::XML::Builder superclass.

tabfile

The tab-delimited file to read. Only one file can be read per TabReader object instantiation.

drug_parent

Optional. Drugs and TestResults can in principle be attached to either Visit or Hospitalisation, and Drugs can further be attached to PriorTreatment. Only one parent class can be supported per run of this script. By default we attempt to attach everything to Visit, but this attribute allows the user to redirect the attachment if necessary.

METHODS

read

Read the tab-delimited file and create the appropriate XML nodes in memory.
