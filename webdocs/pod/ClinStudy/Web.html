NAME

ClinStudy::Web - Catalyst UI for the ClinStudyWeb database

SYNOPSIS

 ./script/clinstudyweb_server.pl

DESCRIPTION

This is the main Catalyst class which drives the ClinStudyWeb user interface.

METHODS

check_model_relationships

Method which checks that there are no objects linked to a given DBIx::Class::Row object via non-cascading_delete relationships. Returns undef if everything is okay, otherwise a string listing the relationships suitable for displaying to the user.

date_today

Returns current date and time in MySQL-friendly format, i.e. "YYYY-MM-DD HH:MM:SS".

recalculate_aggregates

Given a database container object (Visit or Hospitalisation) and a study type CV, runs any applicable test calculator modules according to the configuration file.

form_values_to_database

Wrapper method for $form-model->update($obj)> which tests that the database is not read-only and sets the appropriate journal update information (user, session ID).

update_database_object

Wrapper method for $obj-update()> which tests that the database is not read-only and sets the appropriate journal update information (user, session ID).

delete_database_object

Wrapper method for $obj-delete()> which tests that the database is not read-only and sets the appropriate journal update information (user, session ID).
